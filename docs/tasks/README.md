# Controle de Tarefas (Tasks)

Este diretório contém os checklists de execução para as fases do projeto.

## Status das Tasks:
- [ ] Pendente
- [/] Em progresso
- [x] Concluído

## Fluxo de Trabalho do Agente:
1.  **Ler a Spec correspondente** na pasta `docs/specs/`.
2.  **Marcar a task como "Em progresso"** em `docs/tasks/`.
3.  **Implementar, testar e validar** a funcionalidade.
4.  **Marcar a task como "Concluída"**.

---
*Dica: Crie arquivos por Sprint ou Feature, ex: `task-001-foundation.md`, `task-002-login.md`.*
